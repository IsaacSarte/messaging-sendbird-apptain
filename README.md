# Messaging App with Sendbird

**Technologies Used:**
1. [Next.js](https://nextjs.org/docs) as frontend tech stack
2. [Prisma](https://www.prisma.io/nextjs) for backend using postgresql
3. [bun](https://bun.sh/) as package and runtime manager

### Getting Started

First, install the app:

```bash
bun install
```
Second, run the development server:
```bash
bun dev
```
Incase there are problems regarding database:
- Run the migrations needed first
```bash
bun prisma migrate dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
